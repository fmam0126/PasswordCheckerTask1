# Password Checker Program

![alt text](PseudoCode.png)

### A simple password checker that checks if a password is strong or not.

Tested with: <br>
pass  - BAD password - <u>must</u> be more than 8 characters <br>
password  - BAD password - <u>must</u> be more than 8 characters <br>
passwords  - OK password - more than 8 characters <br>
Passwords  - Better password - more than 8 character and includes an upper case character <br>
Passwords1  - Even better password - more than 8 character and includes and upper case character and digit <br>
Passwords1!  - Good Password - Passes all checks. Upper case. digit. symbol. <br>
